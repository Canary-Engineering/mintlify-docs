## Overview

Flash options are calibration tweaks applied during a flash. Each option modifies specific bytes or tables in the calibration area. Availability depends on engine family, ROM version, and transmission type.

---

## Exhaust & Sound

| Option | Description |
|--------|-------------|
| Burble | Exhaust crackle/pop. Levels: OEM, Softer, Soft, Med, Hard. Per-map-slot. |
| ColdStartReduction | Reduces cold start noise |
| ExhaustFlap | Stock / open in sport / always open |
| StartupRoar | Aggressive engine sound at startup |
| OpfDelete | OPF (particulate filter) delete / overrun fix |
| ShutterFlaps | Intake manifold shutter flap control |
| ShiftBrap | Exhaust pop during full throttle shifts (8AT) |
| ExhaustSetup | OEM vs aftermarket exhaust configuration |

## Option Conflicts

Some options conflict and cannot be used together:

- **BoostPerGear** + **TorquePerGear**: Choose one
- **Burble (Hard)** + **StartupRoar**: Mutually exclusive
- **FlexFuel** variants: One only
- **MultiMap** options: Depends on tune context

The app validates options before flashing and reports conflicts.
