## What is Apex Flashtool?

**Apex Flashtool** is an open-source vehicle ECU toolkit for BMW engines. It provides flash, backup, live monitoring, and diagnostics for DME and TCU control units.

<Columns cols={2}>
  <Card title="Flash" icon="bolt">
    Write calibration tunes to your ECU. Supports OTS maps, custom tunes, and 46+ flash options (burble, vmax removal, flex-fuel, etc.).
  </Card>
  <Card title="Backup & Restore" icon="floppy-disk">
    Create a full calibration backup before flashing. Restore to stock or Stage 0 when needed.
  </Card>
  <Card title="Live Monitoring" icon="gauge">
    Real-time gauges (boost, RPM, AFR), per-cylinder timing, data logging. Export to CSV or .canarylog.
  </Card>
  <Card title="Diagnostics" icon="stethoscope">
    Read and clear DTCs, reset adaptations (12 types), ECU identification, OBD readiness.
  </Card>
</Columns>

## Supported Engines

| Engine | ECU Type | Notes |
|-------|----------|------|
| B58 | MG1 | Bench unlock required |
| S58 | MD1 | Bench unlock required |
| N55, S55 | MEVD17.2 | CRC seed/key |
| N54 | MSD80/81 | KWP2000 (planned) |
| B48 Gen2, S63, N13 | MG1 / MEVD17.2 | As per ECU type |

## Platform Support

- **Android**: Full ECU support via native FFI. Real flashing, backup, monitoring, and diagnostics.
- **iOS / Web**: Mock data for development. Native ECU connection not yet available.

## Safety Disclaimer

<Warning>
**ECU flashing is safety-critical.** This software has not been validated end-to-end on real vehicles/ECUs. Incorrect use can brick an ECU, immobilize a vehicle, or cause mechanical damage. Always create and verify a backup before flashing. Use a battery charger for long operations. The authors are not liable for any damage.
</Warning>
